Normalization

发布于 2020-08-15 4分钟 | 817字数

深度学习Normalization学习

  • 内部协变量偏移(ICS)：在训练深度神经网络的时候，每一层的输入分布会发生变化，这主要是是受到上一层变化的影响。这种现象的存在会减慢训练的速度，对学习率、参数初始化提出一定的要求，而且对具有饱和非线性关系的模型训练十分困难。

Batch Normalization

  • 目的：尽量保证输入分布在随着非线性网络训练的过程中，不会发生太大的偏移，从而使优化器陷入饱和区的概率降低，加速网络的训练。

  • 方法：固定层输入的均值和方差。

  • 两个简化措施：

    1. 不再联合标准化层内的输入、输出特征，而是单独的标准化每个标量特征。
    2. 不再使用整个训练集来计算均值、方差，假设batch能对整体做出较好的估计。

  • 公式：给定mini-batch
    B={x1,x2,...,xm}\mathcal{B} = \{{x_{1},x_{2},...,x_{m}}\}B={x1​,x2​,...,xm​}，其中xi={xi,1,xi,2,...,xi,K}x_{i} = \{{x_{i,1}, x_{i,2},...,x_{i,K}}\}xi​={xi,1​,xi,2​,...,xi,K​}

    μB←1m∑i=1mxi\mu_{\mathcal{B}} \leftarrow \frac{1}{m}\sum_{i=1}^{m}x_{i} μB​←m1​i=1∑m​xi​

    σB2←1m∑i=1m(xi−μB)2\sigma_{\mathcal{B}}^{2} \leftarrow \frac{1}{m}\sum_{i=1}^{m}(x_{i} - \mu_{\mathcal{B}})^{2} σB2​←m1​i=1∑m​(xi​−μB​)2

    x^i←xi−μBσB2+ϵ\hat{x}_{i} \leftarrow \frac{x_{i} - \mu_{\mathcal{B}}}{\sqrt{\sigma_{\mathcal{B}}^{2}+\epsilon}} x^i​←σB2​+ϵ​xi​−μB​​

    yi←γx^i+β≡BNγ,β(xi)y_{i} \leftarrow \gamma \hat{x}_{i} + \beta \equiv BN_{\gamma, \beta}(x_{i}) yi​←γx^i​+β≡BNγ,β​(xi​)

  • 系数γ,β\gamma,\betaγ,β的作用是 the transformation inserted in the network can represent the identity transform ，用来保证特征在标准化的过程中没有丢失原有的信息。

  • Tips:如果输入是白化的(whitened)，例如0均值和单位方差，神经网络的收敛会更快。

Layer Normalization

  • 目的：解决Batch Normalization存在的问题，比如BN与batch size的大小有很大关系，而且它不能应用在类似RNN这种处理动态变长数据的网络结构上。

  • 公式：给定mini-batch
    B={x1,x2,...,xm}\mathcal{B} = \{{x_{1},x_{2},...,x_{m}}\}B={x1​,x2​,...,xm​}，其中$ x_{i} = {{x_{i,1}, x_{i,2},...,x_{i,K}}}$

    μi←1K∑k=1Kxi,k\mu_{i} \leftarrow \frac{1}{K}\sum_{k=1}^{K}x_{i,k} μi​←K1​k=1∑K​xi,k​

    σi2←1K∑k=1K(xi,k−μi)2\sigma_{i}^{2} \leftarrow \frac{1}{K}\sum_{k=1}^{K}(x_{i,k} - \mu_{i})^{2} σi2​←K1​k=1∑K​(xi,k​−μi​)2

    x^i,k←xi,k−μiσi2+ϵ\hat{x}_{i,k} \leftarrow \frac{x_{i,k} - \mu_{i}}{\sqrt{\sigma_{i}^{2}+\epsilon}} x^i,k​←σi2​+ϵ​xi,k​−μi​​

    yi←γx^i+β≡LNγ,β(xi)y_{i} \leftarrow \gamma \hat{x}_{i} + \beta \equiv LN_{\gamma, \beta}(x_{i}) yi​←γx^i​+β≡LNγ,β​(xi​)

Weight Re-scaling and re-centering
